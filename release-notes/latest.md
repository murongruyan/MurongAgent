版本：1.5
版本号：26070715

更新摘要：本次更新完善了模型选择与计费配置体验：应用可从上游读取可用模型，聊天页可直接切换模型和推理深度；同时补齐输入历史持久化，并更新 OpenAI 与 Claude 的默认模型配置。

## 新增

- 新增上游模型目录同步。已配置 API Key 时，可从 OpenAI 兼容接口或 Claude 接口读取模型列表，并与内置推荐和当前模型合并展示。
- 新增聊天输入栏的模型和推理深度快捷切换，选择后会保存为当前 Provider 的默认配置。
- 新增发送历史持久化，最近 50 条非空输入会跨会话保留，可通过输入栏的上下按钮快速回填。
- 新增 OpenAI `GPT-5.6 Sol / Terra / Luna` 与 Claude `Claude Fable 5` 的模型展示、默认推荐和推理提示。

## 优化

- 设置页会显示模型目录来源、同步状态和错误信息，并允许在可选模型列表中直接切换。
- OpenAI 与 Claude 的价格设置改为基于内置官方参考价的倍率；未内置参考价的 Provider 仍可填写绝对价格，成本估算继续使用最终生效价格。
- 自动执行 Profile 会在复杂任务中优先选择新的 OpenAI 和 Claude 推荐模型，同时保留用户自定义网关模型的原有行为。

## 修复

- 修复聊天页自动跟随流式输出时对最后一条消息 ID 的冗余依赖，流式内容更新期间可持续保持预期的跟随行为。

## 稳定性

- 新增 Provider 定价和模型元数据覆盖测试，并同步更新自动执行 Profile 的预期结果。
