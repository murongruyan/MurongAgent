更新摘要：新增审批中心、检查点回滚、记忆库与技能系统，聊天工作流、会话恢复与步骤签收链路全面升级。

## 新增

- 新增审批中心，支持统一查看待审批项、近期审批记录和项目审批状态。
- 新增检查点与回滚中心，支持按代码、对话或全部范围恢复现场。
- 新增上下文压缩提示与压缩历史，长对话可更稳定地继续执行。
- 新增子代理编排视图，可直接查看子代理运行、批次收敛和等待状态。
- 新增 Durable Memory 记忆库，支持全局/项目记忆的保存、检索、读取与清理。
- 新增 Skill System，支持沉淀全局/项目技能并在任务中复用。
- 新增 Doctor 诊断报告导出与崩溃恢复提示，排障更直接。

## 优化

- 聊天工作流升级为计划、澄清、审批、执行统一宿主态，任务推进更稳定。
- 执行计划支持步骤签收与真实工具证据收口，不再只靠口头完成。
- 会话恢复语义增强，旧审批、旧提问、旧子代理在恢复后会被正确标记，减少误操作。
- 项目配置升级为仓库作用域分层保存，项目规则、工具偏好、技能和记忆的边界更清晰。
- 项目编辑器新增 LSP Manager 骨架，多语言诊断链路更统一。
- 更新弹窗现在可直接展示完整版本日志，发版摘要与正文可以从后台和工作流统一回写。

## 稳定性

- 新增最终收口校验，未真正完成的步骤不再容易被误判为已完成。
- 后台任务状态展示更完整，可区分运行中、排队中、整理中、待审批等状态。
- 并行子任务批次现在会显式影响计划阻塞、恢复与证据收口，避免过早完成步骤。
- 会话恢复、检查点、并行子任务、审批链路等多个内核策略已补齐回归保护，整体执行一致性更强。
