# MurongAgent 1.7

版本：1.7
版本号：26071359

## 更新摘要

本次更新重点完善 AI 连接配置与对话执行体验，并提升 Provider 请求、会话持久化和本地文件操作的可靠性与安全性。

## 新增

- **AI 连接配置管理**：支持分别保存、切换、编辑和删除多个 AI 连接；可新增官方 DeepSeek、OpenAI、Claude 配置，以及自定义 OpenAI Compatible / Claude 连接。
- **连接级独立配置**：每个连接可分别保存 API Key、Base URL、模型、推理档位、自动选择选项、价格与余额等信息；切换连接时会一并切换对应配置。
- **对话过程摘要**：将思考、工具调用、文件读取、命令执行、文件修改、子代理与失败状态归纳为可折叠的过程卡片，让最终回答更清晰。
- **聊天内配置选择优化**：在聊天界面中可识别当前选中的连接配置，并更直观地区分模型与思考深度的“自动”或手动选择状态。

## 优化

- **Provider 请求兼容性**：在向 DeepSeek、OpenAI Compatible 和 Claude 发送请求前，自动整理历史工具调用与工具结果，补全被中断的工具调用参数，避免孤立工具消息导致接口拒绝请求。
- **非流式工具调用支持完善**：OpenAI Compatible 与 DeepSeek 的非流式请求现会携带工具定义、工具调用及工具结果上下文。
- **会话历史检索体验**：未填写检索关键词时可返回最近会话；会话筛选参数按标准 JSON 布尔值解析，并统一路径比较规则。
- **工具定义稳定性**：工具 Schema 在注册时缓存并以稳定顺序生成，减少因注册顺序或参数顺序变化造成的请求差异。

## 修复

- **流式响应中断处理**：当流式响应未正常结束时，避免继续执行不完整的工具调用；无有效输出时会作为可重试的连接异常处理。
- **网络重试策略**：改进 HTTP 状态识别与重试逻辑，支持识别 401/403、429、5xx 及 `Retry-After`，并采用带随机抖动的退避等待。
- **会话保存可靠性**：会话、检查点和索引改为临时文件原子写入；取消过期保存任务，避免旧任务覆盖当前会话状态。
- **恢复上下文判断**：修正仅重放的询问状态造成的检查点差异误判，并保留被阻塞或已恢复的最终检查记录作为后续上下文。

## 稳定性

- **本地文件操作边界保护**：代码编辑及文件写入、删除、权限修改均限制在当前项目目录内；无已附加项目或路径越界时会拒绝执行。
- **连接密钥迁移与保护**：旧版单连接配置会自动迁移到新的连接配置结构；各连接的 API Key 继续从持久化配置中剥离并单独安全保存。
- **覆盖关键场景测试**：新增多连接迁移与切换、Provider 消息归一化、工作区路径限制、工具 Schema 稳定性及对话过程摘要相关测试。
